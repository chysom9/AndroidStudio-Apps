
#A collection of Android applications built using Android Studio, demonstrating hands-on experience with mobile UI design, app architecture, and core Android development principles.

Overview

These projects were developed to strengthen practical Android skills, covering a range of app categories from utility tools to interactive user-facing applications. Each app was built with a focus on clean architecture, intuitive UX, and Android best practices.

Technologies Used

Languages: Java, Kotlin
IDE: Android Studio
Build System: Gradle
UI: XML — used to design and build all interactive layouts and user interfaces
